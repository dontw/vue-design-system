# Design System Template

基於 [Vue Design System](https://github.com/viljamis/vue-design-system) 3.5.7 版

**Made by [@viljamis](https://twitter.com/viljamis) and other contributors. See also [the official website](https://vueds.com) of Vue Design System and [read my article](https://viljamis.com/2018/vue-design-system/) on the processes and workflow I use to get started with a new design system project.**

## Requirement

node.js v.10.16.3 以上 (LTS 版本)

## Getting Started

```bash
# clone the project
git clone git@gitlab.halfme.com:frontend-boilerplates/design-system-template.git

# enter the project directory
cd design-system-template

# install dependency
yarn
```

## Build

```bash

# design system 開發模式
yarn start

# 在 /dist build system.css 與 system.js 方便外部引入
yarn build:system
```
